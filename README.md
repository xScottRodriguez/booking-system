## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn
```

## Prepare hooks

### Install husky

```bash
$ yarn run prepare
```

### Generate hook pre-commit

```bash
$ npx husky add .husky/pre-commit 'yarn lint-staged'
```

### Generate hook commit-msg

```bash
$ npx husky add .husky/commit-msg  'npx --no -- commitlint --edit ${1}'
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
